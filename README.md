## Overview

Joplin Professional Suite is a comprehensive open-source note management platform that combines markdown writing, notebook organization, and verifiable encryption for users who want complete control over their note data.

Web clipper and import tools bring content from Evernote exports, web pages, and other note formats into Joplin without losing formatting or attached files during the migration.

A plugin system extends Joplin with community-built tools for templates, table editing, and additional sync backends, adapting the open-source core to individual workflow needs.

Markdown rendering supports extensions including tables, footnotes, and math notation, covering the formatting needs of technical and academic note-taking without proprietary markup.

Geolocation tagging records where a note was created when location services are enabled, adding contextual metadata that can be useful for travel journals or field research notes.

Multi-profile support allows entirely separate note databases — personal and work, for example — to be maintained within the same installation, switching between them without mixing unrelated notebooks.

Joplin Professional Suite is the platform for users who want a note-taking tool whose security claims can be independently verified, with the flexibility to store their encrypted data anywhere they choose.

## Performance & Integration

Joplin is engineered around macOS technologies that handle performance-critical work efficiently. Tasks run in the background without slowing down the rest of your system — you can keep working while Joplin processes operations quietly behind the scenes.

The app integrates deeply with the macOS ecosystem out of the box:

- Launch workflows directly through **Spotlight**
- Preview files instantly with **Quick Look**
- Trigger actions from any app via the **Services menu**
- Fully compatible with **Shortcuts** for automated workflows

- Apple Silicon or Intel — Universal Binary
